1. url_obfuscation_score

Shortining_Service +
double_slash_redirecting +
having_At_Symbol +
Prefix_Suffix

All these features represent URL manipulation techniques used to confuse users.
They are highly correlated and describe the same phishing behavior.


2. ui_deception_score

on_mouseover +
RightClick +
popUpWidnow +

Strong inter-correlation indicates redundancy.


3. trust_score
Formula:
SSLfinal_State +
HTTPS_token +
Google_Index

Reason:
These features indicate legitimacy and trustworthiness of a website.
Combining them creates a stronger trust signal.


4. redirection_risk_score
